通过修改SASS文件夹下config.scss文件相应变量，可以实现便捷灵活控制整个项目的色调、圆角。
核心代码：
$color-main:#00B3D0; //主色
$color-second:#FF7D21; //次色
$border-color:#e5e5e5 ; //边框颜色
$font-color-main:#4b4b4b; //文字主色
$font-color-second:#999999; //文字色
常用色彩
$color-red:red; //红色
$color-blue:#4395FF; //蓝色
$color-green:#55CC55; //绿色
$color-orange:orange; //橙色
组件圆角
$round:r(6);
